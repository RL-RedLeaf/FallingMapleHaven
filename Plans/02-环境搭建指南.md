# 环境搭建指南 — FallingMapleHaven

> 适用平台: Windows 11
> 前置依赖: Docker Desktop、Python 3.11+、Node.js 20+、Git

---

## 1. 基础设施 (Docker)

项目依赖 PostgreSQL 16 + Redis 7，通过 Docker Compose 一键启动：

### 1.1 创建 docker-compose.yml

```yaml
# docker/docker-compose.yml
version: "3.8"

services:
  postgres:
    image: postgres:16
    environment:
      POSTGRES_DB: fallingmaple
      POSTGRES_USER: fm_user
      POSTGRES_PASSWORD: fm_dev_password
    ports:
      - "5432:5432"
    volumes:
      - pgdata:/var/lib/postgresql/data

  redis:
    image: redis:7
    ports:
      - "6379:6379"

volumes:
  pgdata:
```

### 1.2 启动服务

```bash
cd docker
docker-compose up -d
# 验证
docker ps
```

## 2. 后端环境 (Django)

### 2.1 创建虚拟环境

```bash
cd backend
python -m venv .venv
.venv\Scripts\Activate.ps1
```

### 2.2 安装依赖

```bash
pip install django djangorestframework channels channels_redis
pip install django-cors-headers django-environ psycopg2-binary
pip install daphne uvicorn
pip freeze > requirements.txt
```

### 2.3 环境变量 (.env)

```ini
# backend/.env
DEBUG=True
SECRET_KEY=django-insecure-dev-key-change-in-production

ALLOWED_HOSTS=localhost,127.0.0.1
```

### 2.6 asgi.py

```python
import os
from django.core.asgi import get_asgi_application
from channels.routing import ProtocolTypeRouter, URLRouter
from apps.chat.routing import websocket_urlpatterns

os.environ.setdefault("DJANGO_SETTINGS_MODULE", "config.settings.dev")

application = ProtocolTypeRouter({
    "http": get_asgi_application(),
    "websocket": URLRouter(websocket_urlpatterns),
})
```

## 4. 日常开发工作流

```bash
# 终端 1: Docker 服务
cd docker && docker-compose up -d

# 终端 2: 后端
cd backend
.venv\Scripts\Activate.ps1
python manage.py runserver

# 终端 3: 前端
cd frontend
npm run dev
```

### 4.1 重置数据库

```bash
# 清空重建
docker-compose down -v
docker-compose up -d
python manage.py migrate
```

### 4.2 常用命令速查

| 操作 | 命令 |
|------|------|
| 新建 App | `python manage.py startapp <name> apps/<name>` |
| 生成迁移 | `python manage.py makemigrations` |
| 执行迁移 | `python manage.py migrate` |
| 创建管理员 | `python manage.py createsuperuser` |
| Django Shell | `python manage.py shell` |
| 依赖变更 | `pip freeze > requirements.txt` |
| 前端依赖 | `npm install <pkg>` |
