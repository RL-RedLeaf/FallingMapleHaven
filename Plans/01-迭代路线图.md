# 迭代路线图 — FallingMapleHaven

> 版本: v1.0
> 策略: 后端先行 → 前端对接 → 联调验收，每个 Milestone 可独立交付

---

## 总览

```
Milestone 0: 环境搭建       (1d)
     ↓
Milestone 1: 用户系统       (2d 后端 + 1d 前端)
     ↓
Milestone 2: 社交动态       (2d 后端 + 2d 前端)
     ↓
Milestone 3: 个人主页       (1d 后端 + 1d 前端)
     ↓                    ← 同时可开始
Milestone 4: 即时通讯       (3d 后端 + 2d 前端)
     ↓                    ← 依赖 M3 前端插件区
Milestone 5: 互动插件       (2d 全栈)
     ↓
Milestone 6: 圈子与管理     (3d 后端 + 2d 前端)
     ↓
Milestone 7: 扩展与部署     (3d)
```

## 依赖关系图

```
M0 ──→ M1 ──→ M2 ──→ M3 ──→ M5 ──→ M6 ──→ M7
                  ↘                ↗
                   M4 ─────────────┘
```

- M2 好友系统是 M4 私聊的前置
- M3 插件入口区是 M5 展示的前置
- M4 通知系统依赖 M1 用户系统
- M6 小组依赖 M2 动态 + M4 聊天
