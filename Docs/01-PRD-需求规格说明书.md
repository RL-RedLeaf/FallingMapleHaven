## 1. 项目概述

### 1.1 产品定位
面向高中同学及社交圈朋友的轻量社交平台，集个人主页展示、社交互动、即时聊天、趣味工具于一体。

### 1.2 核心目标
- 为同学朋友提供一个私密的线上互动空间
- 每个人都有个性化的个人主页
- 支持实时聊天（私聊+群聊）
- 提供趣味互动工具增进好友关系

### 1.3 目标用户
- 核心用户：高中同学圈子（数十人）
- 扩展用户：社交圈朋友（< 100 人）
- 用户特征：年轻人，熟悉手机操作，追求趣味性和互动感

### 1.4 术语表
| 术语 | 说明 |
|------|------|
| 动态 | 用户发布的图文内容，类似朋友圈 |
| 广场 | 展示所有公开动态的聚合页面 |
| 插件 | 互动工具的统称，如默契问答、匿名提问箱 |
| 小组 | 固定兴趣圈子，有独立页面 |
| 足迹 | 访客访问个人主页的记录 |

## 2. 功能模块详情

### 2.1 用户系统

#### 2.1.1 注册
- **前置条件**：无
- **字段**：用户名（必填，唯一）、密码（必填）、昵称（必填）、邮箱（选填）
- **流程**：填写表单 → 提交 → 系统校验 → 创建用户 → 自动登录 → 跳转到个人主页
- **异常**：用户名重复 → 提示"用户名已被注册"

#### 2.1.2 登录
- **方式**：用户名 + 密码
- **流程**：输入凭证 → 校验 → 设置 Session/Token → 跳转首页
- **异常**：用户名不存在 / 密码错误 → 提示对应信息

#### 2.1.3 个人设置
- 修改昵称、头像、封面背景图、个人签名
- 补充/修改邮箱
- 设置真实姓名 + 是否展示
- 修改密码（需旧密码验证）
- 设置访客足迹是否对外可见

#### 2.1.4 访客足迹
- 任何人访问他人主页时，在对方足迹列表留下记录
- 足迹列表：头像、昵称、访问时间（按时间倒序）
- 用户可设置是否公开自己的足迹列表

### 2.2 好友系统

#### 2.2.1 好友关系
- **状态机**：`无关系 → 已申请 → 已通过(好友)` 或 `无关系 → 已申请 → 已拒绝`
- 双向确认机制：A 申请 → B 收到通知 → B 同意/拒绝
- 好友列表：展示所有已通过的好友

#### 2.2.2 操作
- 搜索用户 → 发送好友申请
- 查看收到的申请 → 同意/拒绝
- 解除好友关系（单向操作，无需对方同意）
- 好友列表按昵称排序

### 2.3 社交广场

#### 2.3.1 动态（核心）
- **发布**：文字（最多 2000 字）+ 图片（最多 9 张）+ 文件附件
- **可见性**：公开 / 仅好友 / 私密（仅自己可见）
- **展示**：头像、昵称、发布时间、内容、图片、文件、互动按钮
- **互动**：点赞（取消点赞）、评论
- **删除**：作者可删除自己的动态

#### 2.3.2 广场页面
- 聚合所有"公开"和"好友可见"的动态
- 按时间倒序排列
- 支持分页/无限滚动

#### 2.3.3 话题板块
- 话题分类（如#闲聊、#求助、#分享）
- 发布动态时可选择话题标签
- 支持按话题筛选浏览

### 2.4 聊天系统

#### 2.4.1 私聊
- 任意用户之间可发起私聊（无需好友关系）
- 仅支持文字消息（首批）
- 消息实时推送（WebSocket）
- 消息列表按最后消息时间排序
- 未读消息数提示

#### 2.4.2 群聊
- 创建群聊：设置群名称 → 选择成员
- 群聊列表展示
- 与私聊共用消息架构
- 群聊功能首批仅含文字聊天

#### 2.4.3 消息存储
- 所有消息持久化存储到数据库
- 消息记录可回溯历史

### 2.5 个人主页

#### 2.5.1 资料卡区域
- 头像、昵称、签名、真实姓名（如设置展示）
- 封面背景图
- 好友按钮（加好友 / 已是好友 / 解除好友）
- 访客足迹入口

#### 2.5.2 动态时间线
- 展示该用户发布的所有动态
- 按时间倒序
- 访问者可见范围受动态可见性控制

#### 2.5.3 插件入口区
- 卡片式布局展示已启用的互动工具
- 首批：默契问答入口、匿名提问箱入口
- 卡片可自定义排列顺序（后期）
- 其他用户可点击卡片进入互动

#### 2.5.4 留言板
- 访客可留言（需登录）
- 主人可回复留言
- 主人可删除留言
- 按时间倒序排列

### 2.6 兴趣小组

#### 2.6.1 小组管理
- 创建小组：名称、简介、封面图、是否公开可见
- 加入小组：公开小组可直接加入 / 申请加入
- 组长权限：修改小组信息、移除成员、转让组长、解散小组
- 成员退出小组

#### 2.6.2 小组页面
- 小组信息展示区
- 小组动态（组内成员发布，仅组内可见）
- 小组聊天室（复用聊天系统）
- **扩展预留**：后续可添加小组投票、小组相册、小组公告等

### 2.7 互动插件

#### 2.7.1 默契问答
- **创建题目**：用户 A 向用户 B 出题（选择题/判断题）
- **作答**：用户 B 收到通知 → 进入答题 → 提交答案
- **匹配度计算**：A 预设正确答案 → B 回答 → 系统比对 → 计算匹配度百分比
- **展示**：在出题人主页展示与好友的匹配度
- **历史**：可查看所有与好友的答题记录

#### 2.7.2 匿名提问箱
- **提问**：任意用户可向目标用户匿名提问
- **回答**：目标用户可公开回答或不回答
- **展示**：已回答的问答对展示在提问箱页面
- **管理**：用户可删除不想展示的问答
- **管理员可见真实提问者**（仅管理员可追溯）

### 2.8 通知系统

#### 2.8.1 通知类型
- 好友申请通知
- 动态点赞通知
- 动态评论通知
- 评论回复通知
- 新消息提醒（未读计数）
- 默契问答邀请通知
- 匿名提问通知

#### 2.8.2 通知页面
- 所有通知按时间倒序排列
- 已读/未读状态区分
- 点击通知跳转到对应内容

#### 2.8.3 角标
- 导航栏显示未读通知总数
- 未读私信数单独显示

### 2.9 管理员后台

#### 2.9.1 用户管理
- 用户列表：搜索、筛选、查看详情
- 用户状态管理：封禁/解封
- 角色管理：普通用户 / 管理员 / 超级管理员

#### 2.9.2 内容管理
- 动态列表：查看所有动态、删除违规内容
- 评论管理：查看/删除评论
- 匿名内容：**显示真实提问者身份**，便于追溯

#### 2.9.3 站点设置
- 注册开关：开放注册 / 仅邀请注册
- 站点公告：编辑并展示在首页
- 网站名称、Logo 等基础信息配置

#### 2.9.4 插件管理
- 查看所有插件列表
- 启停插件（启用/禁用）
- 配置插件参数

#### 2.9.5 数据统计
- 注册用户总数、日活用户
- 动态发布量、评论量
- 聊天消息量

#### 2.9.6 操作日志
- 记录管理员的敏感操作
- 操作人、操作时间、操作内容、操作对象

## 3. 非功能需求

### 3.1 性能
- 页面加载时间 < 2s（正常网络）
- 消息发送延迟 < 500ms
- 支持 100 用户同时在线

### 3.2 兼容性
- 支持 Chrome/Firefox/Safari/Edge 最新版本
- 支持 iOS Safari + Android Chrome

### 3.3 安全
- 密码加密存储（bcrypt/argon2）
- Session/Token 认证
- 文件上传类型限制（图片: jpg/png/gif/webp, 文件: <50MB）
- XSS 防护、CSRF 防护
- 管理员操作日志审计

### 3.4 可维护性
- 模块化插件架构
- 统一的 API 响应格式
- 完整的错误码体系
- 代码注释遵循约定

## 4. 用户故事示例

```
作为一名普通用户
我希望能够发布带图片的动态
并且可以控制谁能看到它
这样我可以分享生活又保护隐私
```

```
作为一名用户
我希望收到好友申请时有通知提醒
并且可以快速同意或拒绝
这样我不会错过社交请求
```

```
作为一名管理员
我希望能够查看匿名提问的真实用户
这样在出现问题时可以追溯处理
```
