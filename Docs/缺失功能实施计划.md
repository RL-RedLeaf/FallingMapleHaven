## Phase 1 — 管理后台 API 客户端与页面接通（已完成）

**目标：** 让现有后台页面从静态壳变为可用页面。

| ID | 描述 | 文件 |
|---|---|---|
| C1-1 | 新增管理后台 API 封装 | `frontend/src/api/admin.js` |
| C1-2 | 接通用户列表与用户状态更新 | `frontend/src/views/admin/AdminUsersPage.vue` |
| C1-3 | 接通动态列表与删除动态 | `frontend/src/views/admin/AdminPostsPage.vue` |
| C1-4 | 接通插件列表与启停插件 | `frontend/src/views/admin/AdminPluginsPage.vue` |
| C1-5 | 接通站点设置读取与保存 | `frontend/src/views/admin/AdminSettingsPage.vue` |
| C1-6 | 接通统计数据 | `frontend/src/views/admin/AdminStatsPage.vue` |
| C1-7 | 接通操作日志 | `frontend/src/views/admin/AdminLogsPage.vue` |

**验收标准：**

## Phase 4 — 插件禁用一致性（已完成）

**目标：** 禁用插件后，不仅主页入口隐藏，直接请求插件 API 也被拒绝。

| ID | 描述 | 文件 |
|---|---|---|
| C4-1 | 增加插件启用状态检查工具或权限类 | `backend/apps/plugins/` |
| C4-2 | 默契问答 API 接入启用状态检查 | `backend/apps/plugins/quizzes/views.py` |
| C4-3 | 匿名提问箱 API 接入启用状态检查 | `backend/apps/plugins/questions/views.py` |

**验收标准：**

- 后台禁用 `quiz` 后，默契问答 API 返回 404 或 403。
- 后台禁用 `question_box` 后，匿名提问箱 API 返回 404 或 403。
- 重新启用后 API 恢复可用。

## 暂不处理

以下内容明确不纳入当前实施计划：

- Dockerfile、Nginx、生产部署文档
- Redis `CACHES`
- Phase 6 功能：纪念日倒计时、许愿池、投票问卷、在线小游戏、音乐分享
- 小组扩展功能：投票、相册、公告
- 前端插件路由动态加载
- `useNotificationStore` / `usePluginStore`
- 大规模组件拆分
