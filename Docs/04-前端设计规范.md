## 4. 响应式设计规范

### 4.1 断点

| 断点 | 宽度 | 设备 |
|------|------|------|
| xs | < 640px | 手机 |
| sm | 640-768px | 大屏手机 |
| md | 768-1024px | 平板 |
| lg | 1024-1280px | 桌面 |
| xl | >= 1280px | 大屏桌面 |

### 4.2 布局适配要点

**手机端 (< 768px):**
- 使用 `MobileTabBar` 底部导航替代顶部 `NavBar`
- 内容区全宽，左右边距 16px
- 卡片圆角 12px，间距 12px
- 聊天输入框吸底
- 管理后台使用顶部 Tab 切换替代侧边栏

**桌面端 (>= 768px):**
- 顶部 `NavBar` 固定
- 内容区最大宽度 960px，居中
- 卡片居中排列，间距 16px
- 管理后台使用侧边栏
- 广场动态可两列瀑布流

**通用规则:**
- 所有按钮触控区域 >= 44px（手机端）
- 图片使用 `object-fit: cover` 自适应
- 表格在手机端改为卡片式列表
- 弹窗在手机端全屏底部弹出

### 4.3 Tailwind 响应式示例

```html
<!-- 导航栏: 手机端隐藏文字, 桌面端显示 -->
<nav class="hidden md:flex">桌面导航</nav>
<div class="flex md:hidden">手机导航</div>

<!-- 两列布局: 手机端单列, 桌面端双列 -->
<div class="grid grid-cols-1 md:grid-cols-2 gap-4">
```

## 5. Pinia 状态管理设计

### 5.1 Store 划分

| Store 名称 | 职责 |
|-----------|------|
| `useAuthStore` | 当前用户信息、登录状态、Token |
| `useFeedStore` | 广场动态列表、话题筛选 |
| `useChatStore` | 聊天列表、WebSocket 连接、消息缓存 |
| `useNotificationStore` | 通知列表、未读数 |
| `useFriendStore` | 好友列表、申请列表 |
| `usePluginStore` | 插件状态、互动数据 |

### 5.2 useAuthStore 示例

```javascript
// stores/auth.js
export const useAuthStore = defineStore('auth', {
  state: () => ({
    user: null,           // 当前用户信息
    isAuthenticated: false,
    loading: false,
  }),
  actions: {
    async login(username, password) { /* 调用API */ },
    async logout() { /* 调用API */ },
    async fetchMe() { /* 获取当前用户 */ },
    async updateProfile(data) { /* 更新资料 */ },
  },
  getters: {
    isAdmin: (state) => state.user?.is_admin || false,
    userId: (state) => state.user?.user_id,
  },
})
```

```javascript
    connectWebSocket() { /* 建立WS连接 */ },
    sendMessage(roomId, content) { /* 发送消息 */ },
    disconnectWebSocket() { /* 断开WS */ },
  },
})
```

## 7. 插件入口卡片规范

每个插件在个人主页中展示为一个卡片，遵循以下接口：

```javascript
// PluginCard.vue 接收
props: {
  plugin: {
    type: Object,
    required: true,
    // { type: 'quiz', name: '默契问答', icon: 'help-circle', 
    //   route: '/profile/1/plugins/quiz', is_active: true }
  }
}
```

插件卡片样式：
- 圆角 12px，纯白背景
- 左侧：SF Symbols / 图标
- 中间：插件名称 + 简要描述
- 右侧：箭头指示可点击
- hover/focus 时轻微上浮阴影
