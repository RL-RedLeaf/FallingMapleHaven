### 1.2 文件类型校验不严

**文件**：`backend/apps/accounts/views.py:148-149, 177-178`  
**严重性**：🔴 严重  
**状态**：✅ 已修复 — 添加 `PIL.Image.verify()` 验证图片真实性，拒绝非图片文件。

```python
# 当前代码（AvatarUploadView / CoverUploadView）

    file.seek(0)
except Exception:
    return api_response(...)
```
